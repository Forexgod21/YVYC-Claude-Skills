# How To Use: Presentation & Pitch Deck Designer

**Category:** Design
**Skill File:** `SKILL.md`
**Original Source:** Alex AI Updates (public Facebook post, 2026)

---

## What This Skill Does

Turns Claude into a pitch deck design director that gives you complete visual
direction for any presentation. Slide layout principles, data visualization styles,
color usage per slide type, typography hierarchy — and the specific design decisions
that separate a forgettable deck from one that looks like a $50,000 agency built it
overnight.

---

## When To Use It

- You're building a pitch deck for investors and need it to look world-class
- Your presentation looks cluttered, generic, or like a default template
- You need to present data in a way that's visually compelling, not just readable
- You're preparing for a high-stakes meeting and the visual quality matters
- You want to hand a designer a precise visual brief instead of a vague direction

---

## How To Activate It

Once installed, activate this skill by saying something like:

> "Design the complete visual direction for a presentation on [topic] for [audience]."

Other ways to trigger it:
- "Make my pitch deck look like a $50,000 agency built it"
- "Give me slide layout and design direction for a [goal] presentation"
- "Create a visual system for my [topic] pitch deck"

---

## What To Provide

| Input | Required? | Example |
|---|---|---|
| Presentation topic | Yes | "YVYC ecosystem overview for investors" |
| Audience | Yes | "Early stage investors and venture partners" |
| Number of slides | No | "12 slides" |
| Slide topics | No | "Cover, problem, solution, market, team, ask" |
| Brand identity | No | "Gold, black, deep purple. Bold and intelligent." |
| Presentation goal | No | "Secure seed funding" |

---

## Example Usage

**You say:**
> "Design the complete visual direction for a 12-slide pitch deck on the YVYC
> ecosystem for early stage investors. Brand colors are gold, black, and deep
> purple. Goal is to secure seed funding. Slides: cover, problem, solution,
> market size, product demo, traction, team, financials, roadmap, competition,
> why now, ask."

**Claude delivers:**
> Complete visual direction for all 12 slides — foundation color and type system,
> layout principles per slide type, data visualization style for the financials
> and market size slides, color assignments per slide category, typography
> hierarchy, and the 5 specific agency-level design decisions that make this
> deck look like it cost $50,000 to produce.

---

## Pro Tips

- The "Agency-Level Design Decisions" section is the most valuable output — implement all of them
- This skill covers visual direction only — use a writing skill for the actual slide content
- If you have an existing deck that needs redesigning, describe what it currently looks like so Claude can improve it specifically
- Works for any tool — PowerPoint, Keynote, Google Slides, Canva, Figma
